# Biblioteca Virtual em Dart

## Descrição

Projeto desenvolvido em Dart para simular uma biblioteca virtual.

A aplicação permite:

* Cadastrar livros
* Remover livros
* Listar livros cadastrados
* Utilizar um menu interativo para acessar as funcionalidades

## Estrutura do Projeto

### Classe Livro

Representa um livro da biblioteca com os seguintes atributos:

* Título
* Autor
* Ano de publicação
* Gênero

### Classe Biblioteca

Responsável por gerenciar os livros cadastrados através dos métodos:

* cadastrarLivro()
* removerLivro()
* listarLivros()

## Tecnologias Utilizadas

* Dart
* Programação Orientada a Objetos (POO)
